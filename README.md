# XTAL Linux

> **Precision. Clarity. Power.**

The official homepage for **XTAL Linux**, a performance-oriented, dark-themed Linux distribution based on **Lubuntu 24.04 LTS**. This Linux distribution is a lightweight, fast, and stable operating system that is perfect for everyday use. It is also a great choice for developers and power users who need a reliable and customizable operating system. It is built with the latest technologies and tools to provide a seamless and enjoyable user experience. XTAL Linux's UI is customized for a aesthetic and user-friendly experience. It is also a great choice for those who want to learn more about Linux and how it works.

### Installation

1. Download the latest version of XTAL Linux from the [official website](https://xtal-linux.github.io)
2. Create a bootable USB drive using [Rufus](https://rufus.ie/) or [balenaEtcher](https://www.balena.io/etcher/)
3. Boot from the USB drive and follow the installation instructions
